iPhone 13

Introduction

L'iPhone 13 est le dernier smartphone d'Apple, connu pour ses performances de pointe, son superbe écran et son appareil photo amélioré.

Caractéristiques

  • Processeur : Puce A15 Bionic
  • Écran : OLED Super Retina XDR de 6,1 pouces
  • Caméras : Système de caméra double, caméra frontale TrueDepth
  • Stockage : Disponible en plusieurs capacités (128 Go, 256 Go, 512 Go)
  • Système d'exploitation : iOS
  • Connectivité : 5G, Wi-Fi 6, Bluetooth 5.0, NFC
  • Batterie : Autonomie améliorée
  • Design : Boîtier en verre et en aluminium, plusieurs finitions et couleurs

Sécurité

L'iPhone 13 utilise la reconnaissance faciale Face ID pour garantir un déverrouillage sécurisé.
